Inicio rápido

Comience incluyendo CSS al archivo html

  1. Crea un nuevo index.html archivo en la raíz de su proyecto. Incluir el <meta name="viewport"> etiqueta también para comportamiento receptivo adecuado en dispositivos móviles. 

    
          <!DOCTYPE html>
          <html>
            <head>
              
              
              CLJ CSS Demo
            </head>
            <body>
              <h1>Hola Mundo! con CLJ CSS</h1>
            </body>
          </html>

  2. Incluya CSS de CLJ CSS. Colocar el <link> etiqueta en el <head> para nuestro CSS 

    
          <!DOCTYPE html>
          <html>
            <head>
              
              
              CLJ CSS Demo
              
            </head>
            <body>
              <h1>Hola Mundo! con CLJ CSS</h1>
            </body>
          </html>

  3. Hola mundo! Abra la página en su navegador de su elección para ver su página con CLJ CSS. Ahora puedes comenzar a construir con CLJ creando el tuyo diseño, agregando docenas de componentes, y utilizando nuestros ejemplos oficiales.
