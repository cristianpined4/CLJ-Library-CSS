Formularios

Ejemplos y pautas de uso para estilos de control de formularios, opciones de diseño y componentes personalizados para crear una amplia variedad de formularios.

Resumen

Los controles de formulario de CLJ CSS se expanden en nuestros estilos de formulario reiniciados con clases. Use estas clases para optar por sus pantallas personalizadas para una representación más consistente en todos los navegadores y dispositivos.

Asegúrese de usar un apropiado type atributo en todas las entradas ( por ejemplo., email para dirección de correo electrónico o number para obtener información numérica ) para aprovechar los controles de entrada más nuevos, como la verificación por correo electrónico, la selección de números y más.

Controles de formulario

Controles de forma textual — <input>s, <select>s, y <textarea>s, están diseñados con la clase .form-input. Se incluyen estilos para apariencia general, estado de enfoque, tamaño y más.

Para entradas de archivo, intercambie el .form-input a .form-input-file.

Y si se le agrega al input la clase .outline tomaria este estilo.

Dimensionamiento

Establecer alturas usando clases como .form-input.lg y .form-input.sm.
